# TalentForge AI Recruitment Platform

AI-powered recruitment analytics platform with resume parsing, candidate ranking, skill matching, recruiter insights, and interactive dashboard visualization.

## Features

- Resume upload and parsing
- AI-based candidate ranking
- Skill matching system
- Analytics dashboard
- Recruiter insights
- Domain-based filtering
- Interactive charts and graphs
- Modern responsive UI

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Vite

Backend:
- FastAPI
- Python
- SQLite
